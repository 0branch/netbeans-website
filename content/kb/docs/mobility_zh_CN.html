什么是 Mobile 应用程序？

大多数 Mobile 应用程序使用 Java Micro Edition (Java ME) 平台，该平台是为移动电话之类的小型设备开发的，但现在可用于多种设备。Java ME 使用向下扩展的 Java SE 组件、虚拟机和 API 所组成的子集。它也定义分别以客户移动设备和嵌入设备为目标的 API。NetBeans IDE 支持 Oracle Java ME 嵌入式和 Java Card 平台。
